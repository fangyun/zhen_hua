北京市振华学校欢迎您的加入

招聘学科及人数

小学：数学、语文、英语、围棋、体育、音乐、美术各科多名教师。

招聘条件

  1. 具有全日制师范类本科及以上学历；
  2. 具有2年以上教学经验（优秀毕业生也可以考虑）；
  3. 愿意致力于民办学校的教学工作；
  4. 有组织学生课外活动的特长，在组织学生参加各类竞赛中获得奖项者优先录用；
  5. 身体健康，个人形象良好；
  6. 岗位职责：小学教育及相关工作

薪资福利

  1. 学校提供丰厚的薪金和奖励（月薪3000—5000元）；
  2. 学校可为教师提供食宿；
  3. 学校可为教师缴纳社保；
  4. 学校为个人提供职业发展、晋升平台。

招聘程序

  1. 应聘者请将本人简介书面材料（注明应聘岗位和联系电话）、学历、职称、教师资格证、身份证、各种获奖证书复印件，近期生活彩照，以电子版的形式发送我校指定邮箱，注明应聘学科。
  2. 经审核入选者，学校将以邮件形式、短信或电话通知具体面试时间，来校面试时，请带上证件原件备查。经试讲、考核入选者，即可聘用（应聘材料恕不退回，同时，本校不负因此产生的任何法律责任）
